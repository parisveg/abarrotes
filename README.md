## Sistema de Gestión de Inventario con Control de Lotes

Este proyecto implementa un sistema profesional de gestión de inventario orientado a tiendas, abarrotes y negocios que requieren controlar productos, su disponibilidad y la fecha de caducidad de cada lote asociado.
El sistema permite mantener un seguimiento preciso del inventario, registrar ventas, identificar productos próximos a caducar y administrar múltiples lotes por producto.

## Funcionalidades Principales
- Gestión de Productos
- Control de Lotes
- Control de Caducidad
- Actualización de Inventario

  ## Objetivo del proyecto
  - El control preciso del inventario en negocios pequeños o medianos.
  - La reducción de pérdidas por productos caducados.
  - La toma de decisiones basada en información actualizada.
  - La organización eficiente del stock a través de lotes.
